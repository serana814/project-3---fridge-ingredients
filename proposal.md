# Mini Project #3 Proposal - Recipe from what is In your Fridge

## What I'm building
A recipe finder where users enter an ingredient they have and see meals they can make with it.

## Which API I'm using
TheMealDB — https://www.themealdb.com/api/json/v1/1/

## Why I chose this
I always stuggle with figuring out how to use all of my produce before it goes bad and having something like this could help me find recipes that I had not thought of so that my produce does not go bad. 

## Core features
1. Ingredient search input
2. Display matching meal cards (photo + name)
3. Click a meal to see full ingredients + instructions
4. Loading state while data fetches
5. Error state if the API fails

## What I don't know yet
- How fetch() and async/await work
- How to read and use JSON data from an API response
- How to dynamically render results with JavaScript
